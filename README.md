## Trader's Journal
трейдерский сервис для анализа паттернов собственных торгов.

#### MVP Functionality
* Добавление в настройки сигналов, бумаг и количества лотов.
* Добавление торговых данных в таблицу. 
* Отображение графиков статистики.
